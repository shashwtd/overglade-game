Overglade
=========

Overglade is a 2D game built using Godot.  
The project focuses on clean mechanics, responsive controls, and smooth gameplay.

Built With
----------
- Godot 4
- GDScript

Getting Started
---------------
1. Clone the repository
2. Open Godot Engine 4
3. Import the project folder
4. Press Play
