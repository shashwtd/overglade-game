AXE HUNTER
=========

Axe Hunter is a 3D game built using Godot.  
The game is fun !!

Built With
----------
- Godot 4
- GDScript

Getting Started
---------------
1. Clone the repository
2. Open Godot Engine 4
3. Import the project folder
4. Press Play
